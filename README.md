# TwitterApiBackProject

## URL
```
127.0.0.1:3000

```

## Launch docker
```
docker-compose up
```

## Access to docker
```
docker exec -ti container bash
```

## register users
```
localhost:3000/users/register
```
## login users
```
localhost:3000/users/login
```
## list users
```
localhost:3000/users/profile
```
